## 🚀 About Me

I'm a passionate developer and **Computer Systems Technology** student at **Sheridan College** (2024-2027), specializing in analysis, design, and full-stack development. I build practical web applications that solve real-world problems, with a focus on educational tools and community-driven projects.

- 🔭 **Currently Building:** [SignalsMaster](https://github.com/mjsamaha/oaksignal-signalsmaster) - helping Sea Cadets master naval signal flags
- 👯 **Next Project:** Cadet Inventory Management System for streamlined logistics
- 🌱 **Learning:** Spring Security patterns, Next.js advanced features, and clean team architectures
- 💼 **Volunteer Work:** Civilian Instructor at Cadets Canada, teaching youth aged 12-18
- 🎯 **Interests:** Minimalist setups, silent coding workflows, CRUD applications
- 🐦 **Fun Fact:** Avid bird watcher and nature photographer
